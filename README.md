# Đề tài: Tìm hiểu và cài đặt thuật toán Naïve Bayes phân loại hoa Iris (Diên vĩ).
**Giảng viên hướng dẫn: Nguyễn Nhứt Lam**<br>
**Sinh viên thực hiện: Thạch Tấn Lộc**

## Mô tả đề tài:
Đề tài tập trung vào nghiên cứu (về mặt lý thuyết) và ứng dụng (về mặt thực hành) thuật toán Naïve Bayes, một phương pháp phân loại kinh điển trong lĩnh vực học máy. Đề tài sử dụng ngôn ngữ lập trình Python kết hợp với các thư viện khoa học dữ liệu (như Scikit-learn, Pandas) để xây dựng một mô hình hoàn chỉnh, có khả năng dự đoán và phân loại các loài hoa trong bộ dữ liệu Iris.
Bằng cách vận dụng các kiến thức cốt lõi về xác suất thống kê và định lý Bayes, đề tài sẽ tập trung xử lý các yếu tố then chốt như: phân tích và tiền xử lý dữ liệu (EDA), lựa chọn và huấn luyện mô hình (model training), và đánh giá hiệu suất của mô hình (model evaluation) thông qua các chỉ số đo lường.
